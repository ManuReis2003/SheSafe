Introdução

Este Storybook serve como uma referência viva para desenvolvedores e designers. Como o aplicativo será desenvolvido em React Native, todos os exemplos de código aqui presentes seguirão os padrões e a sintaxe do React Native.

Notificações e Modais

Componentes para feedback ao usuário, como notificações (toasts) e diálogos de confirmação (modais).

1. Notificações (Toasts)

Usadas para mensagens informativas e de confirmação que aparecem temporariamente.

Uma notificação de EMERGÊNCIA foi enviada para seus contatos definidos!
Uma notificação de RISCO foi enviada para seus contatos definidos!
CONTATO cadastrado com sucesso!

Resumo do Estilo:

  • Fundo: Cinza claro (`#E9E9E9`).
  • Fonte: Montserrat, cor escura (`#343a40`).
  • Destaque: Texto em caixa alta, negrito, com a cor correspondente à ação (vermelho, laranja, verde).
  • Bordas: Arredondadas (aprox. 20px).

Código React Native (Componente Toast)


import React from 'react';
import { View, Text, StyleSheet } from 'react-native';
import { colors } from '../constants/colors';

// A implementação real de um "toast" geralmente requer uma biblioteca
// ou um Provider no topo da árvore de componentes do app.

const ToastView = ({ text, highlightText, highlightColor }) => {
  return (
    <View style={styles.toastContainer}>
      <Text style={styles.toastText}>
        {text}
        <Text style={{ color: highlightColor, fontWeight: 'bold' }}>
          {` ${highlightText.toUpperCase()} `}
        </Text>
        {/* Adicione o resto da mensagem aqui se houver */}
      </Text>
    </View>
  );
};

const styles = StyleSheet.create({
  toastContainer: {
    backgroundColor: colors.toastBg,
    paddingVertical: 20, paddingHorizontal: 25,
    borderRadius: 20, margin: 10, alignItems: 'center',
  },
  toastText: {
    color: colors.dark, fontSize: 16, textAlign: 'center',
    fontFamily: 'Montserrat-Regular', // Exemplo de uso da fonte
  },
});
